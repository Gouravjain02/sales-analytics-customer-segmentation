# Sales Analytics & Customer Segmentation

## Project Overview
End-to-end sales data analysis on 9,789 orders using Python, SQL, and Power BI. 
Identified key business insights and segmented 793 customers using RFM Analysis.

## Key Findings
- West region contributes highest revenue ($710K)
- Technology category leads with $825K in sales
- 50% revenue growth from 2015 to 2018
- 124 Champion customers drive majority of revenue

## Tech Stack
- Python (Pandas, Matplotlib, Seaborn)
- SQL (Pandasql)
- Power BI
- Jupyter Notebook

## Project Structure
- `sales_analysis.ipynb` — EDA + RFM Analysis
- `superstore_dashboard.pbix` — Power BI Dashboard
- `superstore_clean.csv` — Cleaned dataset
- `rfm_data.csv` — RFM segmentation data
